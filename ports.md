# Services, Protocols, Technology Stack, and Ports

This contains the master list of all ports used by the various services in the DNW ecosystem, along with their protocols, technology stacks, and current status. This is a living document and should be updated as new services are added or existing ones are modified.

There is a companion authoritative list named service-hosts.yaml which is used to generate the service-hosts.py, C# class, TypeScript module, and docker-compose.yml files that contain the hostname and port assignments for each service.

Whenever a new service is added or an existing service's port assignment changes, the service-hosts.yaml file should be updated first, and then the code generation scripts should be run to regenerate the relevant files.
